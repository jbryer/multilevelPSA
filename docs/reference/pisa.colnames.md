# Mapping of variables in \`pisana\` with full descriptions.

This data frame provides three variables, \`Variable\` corresponding to
the column names in \`pisana\`, \`ShortDesc\` providing a short
description of the variable as a valid R object name, and \`Desc\`
providing a longer description of the variable.

## Format

a data frame with 50 rows of 3 variables.
