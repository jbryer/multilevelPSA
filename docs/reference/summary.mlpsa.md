# Provides a summary of a `mlpsa` class.

Provides a summary of a `mlpsa` class.

## Usage

``` r
# S3 method for class 'mlpsa'
summary(object, overall.label = "Overall", ...)
```

## Arguments

- object:

  the mlpsa object.

- overall.label:

  the label to place in the strata column for the overall results.

- ...:

  unused.
